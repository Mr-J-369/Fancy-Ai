# Fancy AI 🚀

Fancy AI is a mobile chatbot and generative AI playground built for Android. Inspired by the creative freedom of Local Dream, it bridges cloud intelligence and local networks with mobile creation. Chat with cutting-edge LLMs and generate images using a streamlined mini-A1111 layout right from your phone.

---

## 🔌 Connection Modes
Fancy AI operates as a flexible client-side interface. You can power the app using cloud endpoints or connect it directly to your home server:

* **Cloud APIs (BYOK):** Generate and input your own API keys for **OpenRouter** or **DeepInfra** in the app settings.
* **Local Dream Link:** Connect directly to your local instance over **HTTP** to leverage your own hardware completely free.

---

## ✨ Features

* **Multi-Provider Chat:** Seamlessly connect to OpenRouter or DeepInfra to prompt advanced LLMs.
* **Mini-A1111 Playground:** A lightweight, mobile-optimized image generation suite inspired by Automatic1111.
* **Local Dream Integration:** Hook into your Local Dream HTTP server for private, local generation over your Wi-Fi network.
* **Streamlined UI:** Fast, native Android interface built for immediate prompting and quick iterations.

---

## 🛠️ Getting Started

1. **Download the App:** Grab the latest compiled APK from the [Releases](https://github.com) section.
2. **Setup your Backend:** 
   * **For Cloud:** Grab an API key from [OpenRouter](https://openrouter.ai) or [DeepInfra](https://deepinfra.com).
   * **For Local:** Ensure your Local Dream instance is running and accessible on your network.
3. **Configure the App:** Open Fancy AI, head to settings, enter your cloud keys or your Local Dream HTTP address, and start prompting!
